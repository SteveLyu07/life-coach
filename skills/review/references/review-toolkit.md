## 复盘四步法

当项目失败、目标偏离或犯错后使用。

步骤：

1. **跟随时间，看清过程**：按时间线列出行动、结果和后续影响，只记录事实。
2. **先看做对了什么**：识别成就、内在优势与外部资源。
3. **为偏离辩护**：用“是的，但是……”句式陈述客观原因，还原事实而非自我攻击。
4. **制定下一步计划**：区分偏离类型，输出最小下一步行动。

示例：

```text
是的，我没完成今天的写作目标，但是：
- 我上午处理了三个紧急消息；
- 下午有一个意外会议；
- 我对任务难度的预估不准。

下一步：明天上午先保护 60 分钟写作时间，关掉消息通知。
```

## 成就清单

当用户容易自我否定、长期目标缺乏反馈时使用。

步骤：

1. 每天或每周记录具体成就事件。
2. 注明成就背后的内在优势与外部资源。
3. 定期回顾，看到自己的进步轨迹。

关键提醒：

- 成就不是“大成功”，任何推动进度的小动作都算。
- 重点是“我做了什么让事情变好”，而不是“我还差多少”。

示例：

```text
今天向目标推进的有效动作：
- 把报告大纲列出来了（完成了最难的启动）。
- 和同事确认了数据来源（减少了后续返工）。

这些动作说明我：能启动复杂任务，会主动沟通。
```

## 进度条思维

当长期目标反馈周期慢、用户容易失去动力时使用。

步骤：

1. 为长期目标设计可量化的进度指标。
2. 确定能推动进度条的关键行为。
3. 定期记录成就事件，更新进度。

示例：

```text
目标：三个月内完成行业报告。
进度指标：
- 资料收集：30%
- 大纲完成：10%
- 初稿完成：0%

本周推动进度条的动作：
- 收集了 5 篇核心文献。
- 确定了报告的三个主要章节。
```

## 考官视角

当用户不确定某个方法是否适合自己、方法用起来变味时使用。

核心思想：把自己当作考官，让不同做法“参加考试”，用实际结果判断哪个更适合自己。

步骤：

1. 明确使用某方法的目的。
2. 用 1–2 次小型对比实验测试不同做法。
3. 观察身体感受、行动状态、实际产出。
4. 给做法打分，保留对自己最有效的版本。

示例：

```text
我想测试 25 分钟番茄和 50 分钟长单元哪个更适合写作。
- 周一用 25 分钟番茄：完成了 800 字，但经常被打断思路。
- 周二用 50 分钟长单元：完成了 1200 字，进入心流但休息不够。

结论：写作初稿用 50 分钟单元，修改阶段用 25 分钟番茄。
```

## 预测误差学习

当用户尝试新方法、新习惯或新路线时使用。

核心思想：学习的本质是大脑通过“预测—偏差—校准”减少误差。

步骤：

1. 行动前做出预测：预期会发生什么、会有什么感受、产出多少。
2. 行动后比较实际结果与预测的差异。
3. 分析差异原因。
4. 调整下次预测与行为策略。

示例：

```text
预测：今天早上写 30 分钟能完成 500 字。
实际：只写了 300 字，但思路更清晰了。
差异原因：我以为写作速度是线性的，但实际上初稿阶段思考时间更长。
调整：以后给初稿阶段预留更多思考时间，不只看字数。
```

## 连接意图

当用户行动迷失、被分心带走、不知道当下该做什么时使用。

核心思想：拖延和偏离不是抽象标签，而是“行为偏离了意图”。复盘的重点是持续朝向价值行动。

步骤：

1. 设定清晰意图（发自内心想去的方向）。
2. 在行动中设置检查点，问自己：此刻做的事与意图一致吗？
3. 发现偏离时，不评判，用友善态度把行为拉回。
4. 定期回顾并改进行动。

示例：

```text
本周意图：用更有质量的时间陪伴家人。
检查点：
- 周一晚上一边陪家人一边回消息 → 偏离。
- 周三晚上把手机放远，专注聊天 30 分钟 → 一致。

调整：把“陪伴时间”设为手机勿扰时段。
```

## 以问题为师

当同一类拖延、失败或卡点就反复出现时，把它当作了解自己的入口。

步骤：

1. 列出反复困扰却难以消除的问题。
2. 追问：它折射了我看重的什么？我从中学会了什么？
3. 把学到的流程梳理、情绪调控、优先级判断等技能迁移到其它领域。

示例：

```text
反复问题：每次写公开文章前都拖很久。
它折射了什么：我很在意别人的评价，同时也渴望被看见。
学到了什么：我需要先把“写给自己看”和“发给别人看”分开，先完成草稿再考虑发布。
```

## 总结支点

当用户学了很多方法但不知道用哪个、遇到困难时卡壳时使用。

核心思想：用少数核心概念作为“支点”，遇到困难时快速定位自己所在阶段并找到对应策略。

步骤：

1. 确定自己的核心支点，例如：
   - 朝向价值行动
   - 降低启动阻力
   - 从偏离中学习
   - 保持滋养与消耗平衡
2. 遇到困难时，回到支点判断当前卡在哪一环节。
3. 围绕该支点选择下一步练习或调整。

## 对照实验

当用户想验证某个调整是否有效时使用。

步骤：

1. 明确要测试的变量（只改一个）。
2. 设定观察指标。
3. 试行一段时间（如一周）。
4. 比较试行前后的差异。
5. 决定保留、修改或放弃这个调整。

示例：

```text
假设：早上先处理最重要的任务，而不是先回消息，能提高一天的效率。
变量：早晨前 90 分钟只处理最重要任务。
观察指标：
- 上午完成的重要任务数
- 一天结束时的满足感
- 下午的能量水平

试行一周，周末评估。
```

## 自我关怀式复盘

当用户复盘后更沮丧、自责更强时使用。

核心思想：复盘不是自我检讨，而是带着友善、感恩和欣赏的态度看到自己的进步与努力。

步骤：

1. 在困难时把手放在胸口，做几次深呼吸。
2. 对自己说关怀话语，例如：
   - “谢谢你，努力的你。”
   - “这次没做好，不代表你不好。”
3. 把注意力从“差距”转向“已走的步”。
4. 再问：“此刻我需要什么才能继续前进？”

关键提醒：

- 自我关怀不是自我放纵，而是为有效行动恢复心理资源。
- 只有在情绪被安抚后，人才有能力真正学习和改变。

## 边界

- 这些工具是教练和自我理解用途，不是治疗或医疗建议。
- 当用户有严重情绪困扰或功能障碍时，建议寻求专业支持。
- 不要把复盘变成新的自责工具。如果复盘让用户更痛苦，先回到自我关怀。
