# 人生愿景示例

本示例是公开、虚构、脱敏案例，用来展示 `life-vision` 如何澄清价值观、生成多种未来，并设计一个小实验。

## 用户输入

```text
我工作几年了，收入还可以，大家也觉得这份工作不错，但我越来越觉得没意思。我不是完全讨厌它，就是每天都很像在完成别人规定的动作。我也不知道自己想要什么，只是觉得如果一直这样下去会很空。
```

## 推荐路由

优先使用 `life-vision`。如果用户同时表现出明显耗竭，再合并 `energy-management`。

## 回应示例

```text
我先不把它理解成“你不知足”。你描述的更像是：外部条件还可以，但这份生活没有充分使用到你在意的东西。

我们先不急着问“要不要换工作”，这个问题太大，也太容易把人逼进二选一。更好的起点是找线索：

1. 这份工作给了你什么？
   比如收入、安全感、身份、技能、稳定节奏。

2. 它没有给你的是什么？
   可能是自主、创造、真实关系、成长、贡献感，或者某种“这是我自己选择的生活”的感觉。

3. 过去一年里，有没有三个很小的时刻，让你觉得“这才像我”？
   不一定发生在工作里。可能是一次帮朋友理清问题，一篇写得很顺的文字，一次认真做出来的小作品，或者一次你觉得自己很真实的交流。

从你现在的话里，我先提出一个假设：你不是没有方向，而是你的价值线索还没有被命名。我们可以先把方向写成一个临时版本：

“我想要一种不只是稳定，也能更多表达自主、创造和真实贡献的生活。”

这不是终身誓言，只是下一版设计稿。
```

## 小实验设计

```text
实验名称：两周价值线索实验

想验证的不确定性：
我到底是在厌倦当前工作，还是缺少能表达自主和创造的生活结构？

小行动：
未来两周，每周安排两次 45 分钟，做一件自己选择的小创作或小服务。
可以是写一篇短文、做一个小工具、帮一个朋友解决具体问题，或者整理一个公开笔记。

收集数据：
- 做之前的抗拒程度
- 做的时候是否投入
- 做完后是空、累、满足，还是想继续
- 是否出现“这更像我”的感觉

复盘日期：
两周后。

继续标准：
如果这个实验带来稳定的能量和真实反馈，再考虑把它扩大成一个长期项目。

修订标准：
如果它没有能量，继续找其他价值线索，不急着推翻整个人生方向。
```

## 这个案例展示的能力

- 不急着建议辞职或转行。
- 把抽象空虚转成价值线索。
- 把人生愿景写成临时方向，而不是宏大使命。
- 用低风险实验获得现实反馈。
